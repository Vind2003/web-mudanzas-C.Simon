1. ¿Qué son las cookies?

Una cookie es un pequeño archivo de texto que se almacena en su navegador cuando visita casi cualquier página web. Su utilidad es que la web sea capaz de recordar su visita cuando vuelva a navegar por esa página.

2. Tipos de cookies utilizadas

Este sitio web utiliza las siguientes cookies:

  • Cookies técnicas: Son aquellas imprescindibles para que la web funcione correctamente (por ejemplo, para gestionar el envío del formulario de contacto o la navegación básica).
  • Cookies de terceros: Esta web utiliza servicios de terceros como Google Maps (para mostrar nuestra ubicación) y enlaces a redes sociales (Facebook, Instagram) que podrían instalar sus propias cookies para ofrecerle una mejor experiencia de usuario.

3. Desactivación o eliminación de cookies

En cualquier momento podrá ejercer su derecho de desactivación o eliminación de cookies de este sitio web. Estas acciones se realizan de forma diferente en función del navegador que esté usando:

  • Configuración para Google Chrome
  • Configuración para Safari
  • Configuración para Firefox
  • Configuración para Microsoft Edge

4. Notas adicionales

Ni esta web ni sus representantes legales se hacen responsables ni del contenido ni de la veracidad de las políticas de privacidad que puedan tener los terceros mencionados en esta política de cookies.
